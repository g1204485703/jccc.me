Spring Boot 极简易教程

发表于 2019-02-10

Spring Boot

新建Spring Boot 项目

  1. File - New - Project

  2. 选择 Spring Initializer，使用 Spring 提供的快速构建 Service

    https://start.srping.io

  3. 填写相应项目信息

    Group：组织名称
    Artifact：项目名称
    Version：版本
    Name：项目名

  4. 勾选所需的依赖

  5. 选择本地存储位置

  6. 完成，maven会自动加载依赖，完成项目构建

第一个Spring Boot demo

简单 Controller

  1. 新建 HelloController

  1. 选择 DemoApplication，启动项目，等待项目在8080端口完成启动

  2. 使用浏览器访问 localhost:8080/hello，成功返回 “Hello Spring Boot”
